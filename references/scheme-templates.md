## §1 钻孔桩基础施工方案

### 编制依据（通用）
- 《铁路桥涵工程施工质量验收标准》TB 10415
- 《铁路建设工程施工安全技术规程》TB 10306
- 《铁路工程地基和基础设计规范》TB 10093
- 设计图纸及地质勘查报告

### 第4章 施工工艺与方法（核心）

**4.1 施工工艺流程**
```
测量放样 → 埋设护筒 → 钻机就位 → 钻孔成孔 → 孔径孔深检测
→ 清孔换浆 → 钢筋笼制作 → 下放钢筋笼 → 安装导管 → 灌注混凝土
→ 拔出护筒 → 凿除桩头
```

**4.2 护筒埋设**
- 护筒内径：[桩径+200mm]，壁厚 ≥8mm
- 埋深：旱地 ≥1.5m；水中或流沙地层 ≥2.0m
- 护筒顶高出地面（或水面）[0.3m～0.5m]
- 平面偏位 ≤50mm

**4.3 泥浆制备与控制**
| 参数 | 造孔阶段 | 清孔阶段 |
|------|---------|---------|
| 比重 | 1.05~1.20 | 1.03~1.10 |
| 粘度（s） | 18~22 | 17~20 |
| 含砂率 | ≤4% | ≤2% |
| pH | 7~9 | 7~9 |

**4.4 钻进控制**
- 钻头直径：[设计桩径]，允许偏差 ≥0
- 钻速：[入岩后降速至XXr/min，一般地层XXr/min]
- 进尺速度：[黏土层XX m/h；砂层XX m/h；岩层XX m/h]
- 每进尺2m量一次垂直度

**4.5 终孔与清孔**
- 终孔判定：[地质编录核实入岩深度，嵌岩桩须取岩芯确认]
- 一次清孔：成孔后立即清孔，换浆至合格
- 二次清孔：下完钢筋笼、导管安装后，灌注前再次清孔
- ★ 沉渣厚度：端承桩 ≤50mm；摩擦桩 ≤150mm（高铁：端承 ≤50mm，摩擦 ≤100mm）

**4.6 钢筋笼制作与安装**
- 主筋间距偏差：±10mm；箍筋间距偏差：±20mm
- 笼长偏差：±100mm；笼径偏差：±10mm
- 保护层厚度：[60mm，用滚轮式垫块，间距2m]
- 钢筋笼分节制作时，接头错开布置，同截面接头数量 ≤50%

**4.7 水下混凝土灌注**
- 导管直径：[250mm或300mm]，接头密封良好，提前水压试验
- 导管初始埋深：≥1m，正常灌注保持 1.5～6m
- 导管底口与孔底距离：0.3～0.5m
- 首批混凝土方量：[按公式计算，保证初始埋深≥1m]
- 混凝土坍落度：180～220mm
- ★ 灌注过程中导管不得提出混凝土面，严禁中断超过30min
- 超灌高度：[≥0.8m，凿除浮浆后桩顶标高满足设计]

**4.8 质量检测**
- 成孔质量：逐孔检查孔径、孔深、垂直度、沉渣（填桩基成孔质量检测记录）
- 桩身完整性：低应变动测，检测数量 ≥总桩数30%（普铁）/ 全部（高铁，或遵合同要求）
- 承载力：高应变或静载试验，数量遵设计及合同要求

## §2 承台施工方案

### 第4章 施工工艺与方法

**4.1 施工工艺流程**
```
基坑开挖及支护 → 桩头处理 → 垫层施工 → 承台模板安装
→ 钢筋绑扎 → 预埋件安装 → 隐蔽验收 → 混凝土浇筑（大体积）
→ 养护 → 拆模 → 防水及回填
```

**4.2 基坑开挖**
- 放坡系数：[旱地按1:0.5～1:1.0；水中围堰详专项方案]
- 距桩边距离：承台外边缘 + [施工操作宽度≥0.5m]
- 抽排水措施：[集水坑/管井降水方案]
- 坑底承载力 ≥[设计值]kPa，不符时须换填或加固

**4.3 大体积混凝土温控措施**
- 温控触发条件：最小尺寸 ≥1m
- 控制指标：
  - ★ 中心与表面温差 ≤25℃
  - 表面与大气温差 ≤20℃
  - 降温速率 ≤1.5℃/d
- 测温布置：[每X m一个测温断面，每断面设中心、1/4处、表面三点]
- 散热措施：[预埋冷却水管间距1.0～1.5m，通水冷却]
- 保温措施：[表面覆盖土工布+彩条布，侧模拆除后立即包裹]

**4.4 混凝土浇筑**
- 浇筑方式：[整体分层，斜面推进，分层厚度≤300mm]
- ★ 高铁承台不留施工缝，一次浇筑完成
- 振捣：插入式振动棒，振点间距 ≤500mm，每点振捣时间20～30s


- 模板类型：[圆形钢模/矩形木模/组合模板]
- 分节高度：[单节≤6m（受起重设备限制）]
- 模板拼缝：平整，拼缝间隙 ≤2mm，海绵条嵌缝
- 对拉螺栓：[间距×××mm，φ××对拉杆，端头加100×100×8mm垫板]

**4.3 竖向精度控制**

**4.4 混凝土浇筑**

- 分层厚度：≤300mm（配合振捣棒长度）
- 自由落差：≤2m（超过须用串筒或导管）

## §5 现浇箱梁（支架法）施工方案

> 本章为重点章节，对应危大工程管理要求。

- [支架设计计算书（附件）]

### 第4章 施工工艺与方法

**4.1 施工工艺流程**
```
地基处理 → 支架搭设 → 模板安装 → 支架预压
→ 复测调整 → 底板钢筋绑扎 → 波纹管安装 → 腹板及顶板钢筋
→ 预埋件安装 → 隐蔽验收 → 混凝土浇筑 → 养护
→ 预应力张拉 → 压浆 → 拆模拆架
```

**4.2 支架地基处理**
- 地基承载力要求：[≥XX kPa，依据计算书]
- 处理方式：[换填碎石+夯实/钢管桩/原地夯实]
- 排水：四周设排水沟，防止积水软化地基

**4.3 支架搭设（碗扣/盘扣/门式架）**
- 立杆间距：[横向XX m × 纵向XX m（详见计算书）]
- 步距：[XX m]
- 底座：[可调底座伸出长度≤300mm]
- 剪刀撑：[纵横向每XX跨设一道，与地面夹角45°～60°]
- 扫地杆：距地面 ≤200mm
- 顶部可调托撑：伸出长度 ≤300mm

**4.4 支架预压**
- ★ 预压为强制性要求，预压荷载 ≥1.05倍施工荷载（含混凝土自重+施工活载）
- 预压材料：[砂袋/预制块/注水水箱]，分[X]级加载，每级加载后稳压观测
- 观测点布置：[跨中、L/4截面，横向每XX m一点，共XX点]
- 合格标准：连续24h沉降差 ≤1mm（或连续3天 ≤2mm，具体遵JTG/T 3650）
- 弹性变形、非弹性变形记录，预拱值 = 弹性变形 + 设计预拱值

**4.5 混凝土浇筑顺序**
- 浇筑顺序：底板 → 腹板 → 顶板，对称、均匀推进
- 分层厚度：≤300mm
- 施工缝设置：[一次性浇筑完成，不留纵向施工缝]
- 跨中浇筑完成时间：同一跨混凝土须在初凝前完成

**4.6 预应力施工**
- 张拉条件：混凝土强度 ≥[设计值]（不低于C40），弹性模量达设计值的90%
- 张拉程序：0 → 初应力（20%σcon）→ σcon（持荷2min量伸长量）→ 锚固
- ★ 超张拉量：≤设计张拉力的[5%]，总张拉力允许偏差 ≤±5%
- 张拉记录：张拉力、伸长量双控，实测伸长量偏差 ≤±6%

## §6 现浇箱梁（悬臂浇筑法）施工方案

### 第4章 施工工艺与方法

**4.1 施工工艺流程**
```
0号块施工（托架法）→ 挂篮安装调试 → 对称悬浇1号块
→ 张拉顶板束 → 挂篮前移 → 对称悬浇2号块 → ……
→ 边跨现浇段 → 合龙前配重 → 合龙段浇筑 → 体系转换
→ 二期恒载施加 → 成桥
```

**4.2 挂篮设计要点**
- 挂篮自重 ≤浇筑混凝土重量的0.5（降低不平衡弯矩）
- ★ 挂篮使用前须进行1.2倍设计荷载预压，变形稳定后方可使用
- 挂篮行走：两侧同步前移，速度差 ≤50mm

**4.3 合龙施工**
- 合龙顺序：先边跨，后中跨（连续梁）
- 合龙时间：宜选在一天中温度最低（且稳定）时段，通常凌晨2～5时
- 合龙配重：通过计算确定，两端配重误差 ≤2%
- ★ 合龙段混凝土：采用微膨胀混凝土，强度等级 ≥C50，一次浇筑完成，不留施工缝
- 合龙段浇筑后保湿养护 ≥14d

## §7 钢桁拱转体施工方案

> 超危大工程，须专家论证。

### 第4章 施工工艺与方法

**4.1 施工工艺流程**
```
转体结构基础施工 → 下转盘施工（球铰安装）→ 上转盘施工
→ 拼装胎架搭设 → 主桁杆件拼装（两侧悬臂）→ 配重计算与施加
→ 称重试转 → 正式转体（连续或步进）→ 就位锁定
→ 合龙段施工 → 体系转换
```

**4.2 球铰安装**
- 球铰安装精度：中心位置偏差 ≤2mm；高程偏差 ≤1mm；水平度 ≤0.5‰
- 球铰表面：涂抹四氟乙烯粉润滑，摩擦系数 ≤0.03

**4.3 配重计算**
- ★ 转体前须称重试验，确定不平衡力矩，偏心距 ≤10mm（TB/T 3548规定≤1cm）
- 配重材料：[砂袋/混凝土块]，分级加减至偏心距合格

**4.4 转体施工**
- 牵引设备：[连续千斤顶/步进千斤顶]，功率冗余系数 ≥1.5
- 转体速度：[≤0.01rad/min]（视桥型而定）
- 实时监控：偏转角度（测量机器人全程跟踪）、主梁标高、牵引力
- ★ 转体就位偏差：轴线 ≤5mm；高程 ≤3mm
- 就位后立即临时锁定（∑≥4个楔块），混凝土封铰前不得拆除

**4.5 涉铁施工安全**
- ★ 转体过程中既有线须封锁行车，下穿既有线净空 ≥[XXX]mm（含防护设施高度）
- 防坠落网：梁底以下1m处设兜底安全网

**4.4 天窗施工计划**
- 天窗类型：[垂直天窗/综合天窗/封锁天窗]
- 天窗时间：[由铁路局统一下达，一般XX:XX～XX:XX]
- 天窗内工序：[仅在天窗内完成的关键工序列表]
- 进入天窗程序：[申请→准备→接令→作业→完工清场→销点]
