### 一、施工准备

1. 钻机进场前检查：发动机、卷扬机、钻头、泥浆泵运转正常，液压系统无渗漏。
2. 泥浆池提前挖好，容积不小于桩孔体积的1.5倍；废浆池隔离设置，防止污染。
3. 现场配备足够的泥浆材料（膨润土、纯碱等）。
4. 钻孔前复测桩位，偏差 ≤20mm 后方可埋设护筒。
5. 混凝土浇筑前，确认水下混凝土配合比通知单已签发。

### 二、作业条件

1. 桩位测量复核完毕，轴线已引出至稳定点。
2. 护筒已按规范埋设，孔口护筒标高已记录。
3. 泥浆比重仪、粘度计、含砂率测定仪等检测工具已到位。
4. 导管已检验：水压试验（1.5倍孔深水压）合格，导管长度配置满足孔深要求。

### 三、操作工艺

**3.1 工艺流程**

测量放样 → 埋设护筒 → 泥浆制备 → 钻机就位 → 钻进成孔 → 终孔验收 → 一次清孔 → 下钢筋笼 → 安装导管 → 二次清孔 → 灌注水下混凝土 → 拔出护筒

**3.2 关键操作要点**

（1）**护筒埋设**：护筒顶高出地面（或水位）不小于300mm，护筒外侧用黏土分层夯填密实，防止孔口坍塌。

（2）**钻进**：保持泥浆液面稳定，始终高于地下水位不少于1.0m（水中桩高于水面不少于1.5m）。进入砂层或流砂层时，适当加大泥浆比重（1.15～1.20），降低钻速。

（3）**终孔**：达到设计深度且嵌岩深度满足要求后停钻，立即联系现场工程师进行终孔验收（孔径、孔深、垂直度）。

（4）**清孔**：一次清孔使用正循环或反循环换浆，至孔底沉渣满足要求。二次清孔须在钢筋笼下完、导管安装后，混凝土浇筑前30min内完成。**沉渣厚度：端承桩 ≤50mm；摩擦桩 ≤150mm（高铁摩擦桩 ≤100mm）。**

（5）**钢筋笼安装**：起吊须四点悬挂，防止变形；分节安装时焊接或机械连接须符合规范；钢筋笼就位后立即固定，防止下沉。

（6）**导管安装与混凝土灌注**：导管距孔底0.3～0.5m；首批混凝土量须足以埋住导管底口不少于1.0m；灌注全程保持导管埋深1.5～6.0m；**严禁提管过快露出混凝土面**；超灌高度不小于0.8m。

### 四、质量标准

| 检验项目 | 允许偏差 | 检验方法 |
|---------|---------|---------|
| 桩位（成孔后） | ≤D/6，且≤100mm | 测量 |
| 孔径 | +100mm，-0 | 钻头尺寸/超声波 |
| 孔深 | +300mm，-0（不得小于设计值） | 测绳 |
| 垂直度 | ≤1% | 测斜仪/超声波 |
| 沉渣厚度 | 端承桩≤50mm；摩擦桩≤150mm | 测锤 |
| 桩身完整性 | Ⅰ类或Ⅱ类桩 | 低应变动测 |

### 五、安全注意事项

1. **护筒旁不得站人**，成孔过程中孔口须设防护盖板。
2. 卷扬机钢丝绳每次作业前检查，断丝超标立即更换。
3. 夜间施工须保证照明，孔口设警示灯。
4. 混凝土运输车进场行驶路线须离孔口 ≥2m，防止压塌孔壁。
5. 导管拆卸时防止混凝土飞溅，人员站侧面操作。
6. 泥浆不得直接排入农田或河流，须经沉淀处理后排放。

### 六、成品保护

1. 桩头凿除后，裸露钢筋用塑料薄膜包裹防锈，待承台施工时再打开。
2. 桩孔位置设明显标志，防止施工机械碾压。
3. 冬期施工：混凝土浇筑后孔口保温覆盖，防止顶部混凝土受冻。

## §2 承台施工技术交底

### 三、操作工艺

**3.1 工艺流程**

基坑开挖 → 桩头处理 → 垫层浇筑 → 模板安装 → 钢筋绑扎 → 预埋件安装 → 隐蔽验收 → 大体积混凝土浇筑 → 养护 → 拆模

**3.2 关键操作要点**

（1）**桩头处理**：用风镐或空压机凿桩头至设计标高，严禁用挖机直接冲撞。凿除后桩顶面平整，嵌入承台长度满足设计要求（一般为桩径）。

（2）**模板安装**：大型承台采用组合钢模板，模板竖向接缝用密封条填塞；模板外侧设拉杆和斜撑，确保侧向稳固；模板内壁涂脱模剂。

（3）**钢筋绑扎**：底层钢筋用混凝土垫块（不少于4点/m²）控制保护层厚度；绑扎顺序由下至上，主筋锚固长度满足设计；插入桩顶的钢筋与桩主筋焊接连接时，焊缝长度须满足规范要求。

（4）**大体积混凝土温控**：提前编制温控方案，预埋冷却水管（间距1.0～1.5m，管径φ25～φ32）；开始通水时间：混凝土浇筑后12～24h；通水持续至温差满足要求方可停水；测温孔须同步布设，每4h记录一次。

（5）**浇筑**：采用分层斜面推进法，层厚 ≤300mm；全程对称浇筑，防止模板侧压力不均匀失稳；全面振捣，振动棒插点间距 ≤500mm，每点振捣 20～30s，不得漏振。

### 四、质量标准

| 检验项目 | 允许偏差 |
|---------|---------|
| 平面中心位置 | ±20mm |
| 顶面高程 | ±10mm |
| 截面尺寸 | ±20mm |
| 混凝土强度 | 不低于设计值 |
| 混凝土表面 | 无蜂窝麻面（面积<总面积0.5%为一般缺陷） |

### 五、安全注意事项

1. 深基坑作业须设上下通道（安全爬梯），严禁攀爬模板进出。
2. 基坑边缘堆料距坑边 ≥1.5m，堆高 ≤1.5m。
3. 振捣棒电源线绝缘良好，漏电保护器正常；操作人员戴绝缘手套。
4. 大体积混凝土浇筑连续作业须安排轮班，防止间歇超过混凝土初凝时间。

## §3 墩柱施工技术交底

### 三、操作工艺

**3.1 工艺流程**

测量放线 → 翻模（或整体钢模）安装 → 钢筋绑扎 → 预埋件安装 → 隐蔽验收 → 混凝土浇筑 → 养护 → 脱模 → 外观缺陷修补

**3.2 关键操作要点**

（1）**模板精度控制**：每节模板支设完成后，用线锤或全站仪复测轴线和垂直度；纠偏量 ≥5mm 时须重新调整，不得强行扭曲模板就位。

（2）**翻模作业**：提升模板前，混凝土强度须 ≥1.5MPa（约出机后12～18h）；提升过程匀速，两侧同步；翻到位后立即紧固对拉螺栓。

（3）**混凝土自由落高**：墩高 ≤3m 时可直接入模；超过3m 须用串筒或导管输送，防止骨料分离。

（4）**施工缝处理**：新混凝土浇筑前，施工缝处凿毛（露出骨料），清除浮浆，冲洗湿润，铺设10～15mm 厚同配比水泥砂浆后再浇混凝土。

### 四、质量标准

| 检验项目 | 普铁偏差 | 高铁偏差 |
|---------|---------|---------|
| 轴线偏位 | ±10mm | ±5mm |
| 顶面高程 | ±10mm | ±5mm |
| 垂直度 | H/1000，≤20mm | H/3000 |
| 截面尺寸 | ±20mm | ±10mm |

### 五、安全注意事项

1. 高空作业（≥2m）须系安全带，安全带挂在高于操作面的可靠点。
2. 模板提升时，下方严禁站人，设隔离警戒区。
3. 混凝土泵管接头须牢固，泵送前试压，防止管道爆裂伤人。

## §4 架梁施工技术交底

### 三、操作工艺

**3.1 工艺流程**

桥位测量复核 → 支座安装 → 架桥机纵移就位 → 横移对位 → 梁片运至喂梁区 → 起吊 → 横移落梁 → 就位调整 → 检测验收 → 架桥机过孔

**3.2 关键操作要点**

（1）**架桥机纵移**：行走前检查轨道铺设质量（直线度、高差、扣件紧固），纵移速度 ≤5m/min，两侧同步偏差 ≤20mm。

（2）**梁片起吊**：起吊前确认梁片重量与起吊设备额定载重的匹配；吊索角度 ≥60°（防止梁片受过大侧向压力）；起吊过程匀速，风速 ≥6级停止作业。

（3）**横移落梁**：横移到位后，缓慢下放，两端同步，高差 ≤20mm；梁片与支座接触后，确认支座居中，再完全落梁。

（4）**就位调整**：测量梁端位置，纵向偏差 ≤10mm；横向偏差 ≤5mm；不得用撬棍强行移动已落梁，须起吊重新就位。

（5）**涉越既有线**：在既有线天窗内完成架梁，作业前防护员到位，接令后方可开始；天窗结束前完成清场，确认机械设备退出安全距离。

### 四、质量标准

| 检验项目 | 允许偏差 |
|---------|---------|
| 梁端就位（顺桥向） | ±10mm |
| 梁端就位（横桥向） | ±5mm |
| 支座偏位（纵向） | ≤5mm |
| 支座偏位（横向） | ≤3mm |
| 相邻梁顶面高差 | ≤5mm |

### 五、安全注意事项

1. ★ 架梁全程架桥机下方及梁端投影下方严禁站人，设隔离绳围栏。
2. 梁片运梁车行走路线须经过桥梁承载力验算，限速 ≤10km/h，转弯 ≤5km/h。
3. 架桥机行走前确认前方轨道无障碍物，专人引导。
4. 作业人员须佩戴安全帽、安全带（高处操作时）。

## §5 满堂支架现浇箱梁技术交底

### 三、操作工艺

**3.1 工艺流程**

地基处理 → 支架搭设 → 底模安装 → 支架预压 → 调整底模标高（含预拱） → 底腹板钢筋绑扎 → 预应力管道安装 → 顶板钢筋 → 隐蔽验收 → 混凝土浇筑 → 养护 → 预应力张拉 → 压浆 → 拆模拆架

**3.2 混凝土浇筑要点**

（1）浇筑顺序：底板全幅浇筑 → 腹板（对称，从一端向另一端推进）→ 顶板（最后合拢）。禁止先浇一侧腹板完成后再浇另一侧，防止支架不均匀侧倾。

（2）各部位分层浇筑：底板分两层（每层 ≤150mm）；腹板分层 ≤300mm；顶板一次浇完。

（3）箱内须设作业人员进行振捣，配置不少于[X]台振捣棒；顶板振捣须平板振动器+插入式配合。

**3.3 预应力张拉要点**

（1）张拉前确认：混凝土强度报告合格（≥设计值，且≥C40）；同条件养护试件龄期 ≥[X]天。

（2）两端同步张拉，两端伸长量偏差 ≤10%；总伸长量与理论值偏差 ≤±6%。

（3）张拉顺序：竖弯束 → 顶板纵向束，按设计图纸顺序进行。

（4）及时填写张拉记录，超过允许偏差须立即停张，查明原因。

### 五、安全注意事项

1. 支架搭设须持证人员进行（架子工证），搭设完成须经验收合格。
2. 预压加载须均匀对称，不得集中堆载，加载速度 ≤[X]t/h。
3. 张拉时，千斤顶后方严禁站人（钢绞线崩断危险），在张拉轴线两侧斜向操作。
4. 箱梁内作业须设通风措施（电动轴流风机），防止 CO₂ 积聚。
5. 拆架时从上至下逐层拆除，不得直接推倒。

## §6 悬臂浇筑箱梁技术交底

### 三、操作工艺

**3.1 工艺流程（单个T构）**

0号块（托架法）→ 挂篮拼装调试 → 对称1号块绑筋/立模/浇筑 → 张拉顶板束 → 挂篮前移 → 2号块…以此类推 → 边跨合龙 → 中跨合龙

**3.2 合龙段施工要点**

（1）选择一天中气温最低且稳定的时段（通常凌晨2～5时），混凝土入模温度 ≤[25℃]。

（2）合龙前在两侧悬臂端施加临时压重（用砂袋），重量按计算确定；完成合龙段混凝土浇筑后立即卸去压重。

（3）合龙段钢筋绑扎须与两侧主梁钢筋搭接完整，焊接或套筒连接均须验收。

（4）合龙段混凝土（微膨胀 C50）须一次连续浇筑，不留施工缝。

（5）合龙段养护：立即覆盖湿润草帘，持续保湿养护 ≥14d。

### 五、安全注意事项

1. ★ 挂篮悬挂于已浇梁段，严禁超载；每节段浇筑前复核挂篮悬挂系统状态。
2. 高空临边作业（桥面边缘）须设临时防护栏杆（高度 ≥1.2m）。
3. 挂篮前移时，无关人员撤离悬臂区域，专人监控。
4. 合龙段施工夜间作业须充足照明（≥[200]lux），配备对讲机。

## §7 钢桁拱转体技术交底

> 超危大工程，本交底须在专家论证方案批准后方可组织实施。

### 三、操作工艺

**3.1 转体流程**

配重称重计算 → 砂箱临时支撑检查 → 牵引系统安装调试 → 空转试验 → 按指令正式转体 → 就位微调 → 临时锁定 → 合龙段施工 → 封铰

**3.2 转体前准备**

（1）称重试验：在转体前对悬臂两端进行称重，计算不平衡力矩及偏心距，偏心距 ≤10mm 方可转体；超过须加减配重至合格。

（2）通信系统：转体总指挥（现场）、测量组、牵引操作组、既有线防护员须实现对讲通信畅通，规定信号口令（"准备"、"转"、"停"、"刹"）。

（3）与铁路局协调：确认封锁令已批准，防护员持令上岗。

**3.3 转体过程控制**

（1）起步转体：初始摩阻力最大，千斤顶出力最大，起步时缓慢加压，不得猛冲。

（2）匀速转体：速度控制在 ≤[0.01rad/min]；测量组每旋转1°报告偏差一次。

（3）就位制动：距就位位置[5°]时降低转速，距[0.5°]时停转，用手动千斤顶微调至偏差 ≤5mm。

（4）就位锁定：立即打入楔块（不少于4个），并回顶砂箱支撑，防止晃动；合龙段浇筑完成并达强度前不得拆除临时锁定。

### 五、安全注意事项

1. ★ 转体过程中，既有线封锁执行，防护员不得离岗。
2. 转体范围内设隔离区，严禁无关人员进入。
3. 千斤顶操作人员须持证，转体全程设专人监控油缸压力。
4. 准备停转装置（紧急制动）：牵引钢绞线可随时夹片锁定。
5. 雨天、风速 ≥6级禁止转体。

## §8 钢桁拱顶推技术交底

### 三、操作工艺

**3.1 顶推流程**

主桁拼装 → 安装导梁 → 第1次顶推（1个节间）→ 拼装下一节间 → 顶推… → 导梁接触对岸临时墩 → 继续顶推至设计位置 → 拆除导梁 → 落梁就位

**3.2 关键要点**

（1）顶推装置：[水平千斤顶+锚块+顶推梁]或[连续顶推器]，行程与节间长度匹配。

（2）导梁作用：减小顶推前端的悬臂弯矩，每次前端搭上支墩后，弯矩大幅减小，可继续顶推。

（3）线形控制：每步顶推完成后测量标高和轴线，偏差超过20mm（横向）或30mm（竖向）须停止检查原因。

（4）支墩滑板：在支墩顶部安装四氟乙烯滑板，定期补涂润滑脂，保证顶推摩阻系数 ≤0.05。

### 五、安全注意事项

1. 顶推行走时，主桁下方严禁站人。
2. 液压系统须定期检查，保压能力良好；每次顶推前检查各油缸、管路接头。
3. 跨越既有线顶推须在封锁天窗内完成，并设防坠落网（净空下1.0m处）。

## §9 涉铁路基及过渡段技术交底

### 三、操作工艺

**3.1 过渡段填筑流程**

原地面清理 → 地基处理（换填/碾压） → 测量分层标识 → 卸料摊铺 → 整平 → 碾压 → 压实度检测 → 合格后进入下一层

**3.2 关键操作要点**

（1）**台阶接坡**：过渡段与相邻路堤相接处开台阶，台阶高 ≤500mm，台阶面向内（向路基中心）倾斜 ≥2%，防止积水。

（2）**分层填筑**：每层松铺厚度 ≤200mm，压实后 ≤150mm；级配碎石须均匀摊铺，不得集中大粒径骨料。

（3）**碾压程序**：先用振动压路机静压1遍，再振动碾压 ≥6遍（根据试验段确定），从两侧向中间碾压，相邻碾压轮迹重叠 ≥200mm。

（4）**涉铁防护**：施工机械在既有线安全距离（≥3.2m）以外作业；需在安全距离内作业时，须申请天窗封锁，防护员接令后方可作业；作业完成后施工机械立即退出安全距离，清场销点。

（5）**封锁天窗作业程序**：
- 提前[X]天提报天窗申请
- 天窗当日：[XX:XX] 前防护员到位 → 接令 → [XX:XX] 开始作业
- 作业完成：清查人员、机具，确认安全后向调度汇报 → 销点 → [XX:XX] 开通

### 四、质量标准

| 检验项目 | 高铁标准 | 普铁标准 |
|---------|---------|---------|
| 压实系数 K | ≥0.97 | ≥0.95 |
| 过渡段长度偏差 | ±100mm | ±200mm |
| 顶面宽度偏差 | ±50mm | ±100mm |
| CBR值（基床表层） | ≥60% | ≥40% |

### 五、安全注意事项

1. ★ 未经铁路局批准，严禁任何人员和机械进入既有线安全防护距离内。
2. ★ 防护员须持证上岗，防护区间内列车通过前发出"注意来车"口令，所有人员停工避让至安全距离以外。
3. 翻斗车在既有线附近行驶须限速 ≤5km/h，严禁急转弯或急刹车。
4. 夜间施工须照明充足，安全帽佩戴反光条，背心穿反光材料。
5. 台风、大雨等极端天气停止施工。
