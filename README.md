## 功能概览

| 功能 | 说明 |
|------|------|
| 📋 **施工方案生成** | 自动输出含编制依据、工程概况、施工工艺、质量控制、安全管控的完整方案框架 |
| 📝 **技术交底编写** | 工班级 / 专项级技术交底，含工艺流程、操作要点、质量标准、安全事项 |
| 📖 **规范条文引用** | 自动定位 TB / JTG / GB 系列规范条款，强制性条文（★）显著标注 |
| 🔢 **力学验算** | 支架、模板、球铰、贝雷梁、桩基等临时结构验算，公式+参数+结论一键输出 |

## 使用方式

### 在 WorkBuddy / Claude 中安装

```bash
npx skillhub install www17growcom/tb-bridge-construction
```

### 触发词（对话中说出以下关键词即可激活）

- **方案类**：施工方案、专项方案、超危大方案、安全专项
- **交底类**：技术交底、安全交底、工艺交底
- **规范类**：规范要求、条文规定、验收标准
- **验算类**：验算、承载力计算、稳定性计算、力学计算
- **工艺词**：桩基、承台、墩柱、架梁、现浇梁、钢桁拱、转体、顶推、过渡段、涉铁施工

### 使用示例

```
用户：帮我写一个钻孔桩施工方案框架，普速铁路，桩径1.5m，桩长35m

用户：写满堂支架现浇箱梁的技术交底，支架高度12m，跨度40m

用户：验算对拉螺栓，φ16，间距600×600mm，墩柱截面2.5m×2.5m，浇筑速度1.5m/h

用户：TB 10415 对钻孔桩沉渣厚度有什么要求？
```

## 注意事项

1. **规范时效**：本 Skill 收录规范均为最新版本（截至 2024 年），使用前请核实规范版本是否有更新。
2. **验算结果**：AI 输出的验算结果仅供参考，正式报审文件须由注册工程师复核签字。
3. **超危大工程**：转体（≥3000t）、顶推（≥60m）、高支模（≥8m）等须经专家论证，本 Skill 辅助编制方案框架，不替代专家论证程序。
4. **涉铁施工**：所有涉及既有铁路的施工方案，须经铁路局审批，本 Skill 内容仅作技术辅助。
